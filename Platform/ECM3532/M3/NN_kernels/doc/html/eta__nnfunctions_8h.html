Q15 sigmoid activation function using direct table look-up.

Parameters
[in,out]datapointer to input
[in]sizenumber of elements
[in]int_widthbit-width of the integer part, assume to be smaller than 3

This is the direct table look-up approach.

Assumption: the integer part of the fixed-point is <= 3. More than 3 doesn't make much sense, as there is no difference with saturation followed by any of these activation functions.
